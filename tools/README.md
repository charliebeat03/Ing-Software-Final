# Tools

## maintenance
Scripts manuales de limpieza, migracion o reseteo de base de datos.

## diagnostics
Pruebas tecnicas y scripts para detectar problemas de carga o estructura.

## installer
Archivos heredados para compilacion, instalador y empaquetado.

Estos recursos se movieron fuera de la raiz para que el flujo principal de la app quede mas claro.
